# 欢迎使用 LinkBox

LinkBox 是一个强大的链接管理工具，帮助您更好地组织和管理您的网络资源。

## 主要功能

- 📑 收藏服务 - 便捷地保存和管理您的网页链接
- 📁 收藏集 - 创建个性化的链接集合
- 🏷️ 标签系统 - 灵活地组织和分类您的链接
- 🔍 智能搜索 - 快速找到您需要的内容
- 🤖 AI 建议 - 智能推荐相关内容
- 🗑️ 回收站 - 安全地管理已删除的内容
